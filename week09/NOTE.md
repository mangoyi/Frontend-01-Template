# 重学动画
Animation

1. @keyframes 定义

@keyframes myKf{
    from {
        background: red;
    }
    to {
        background: yellow;
    }
}

div {
    animation: myKf 5s infinite;
}

1. animation-name 时间曲线
2. animation-duration 动画的时长
3. animation-timing-function 动画的时间曲线
4. animation-delay 动画开始前的延迟
5. animation-iteration-count 动画的播放次数
6. animation-direction 动画的方向
